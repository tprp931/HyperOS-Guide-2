🔧 Розблокування завантажувача

Xiaomi — швидкі кроки

  1. Налаштування → Про пристрій → 7 разів "Номер збірки", потім увімкнути OEM Unlock і USB Debugging.
  2. Використати Mi Unlock Tool на ПК.
  3. Fastboot режим → підключити до ПК → Run Mi Unlock → Unlock (72–168 годин).

Увага: стирає всі дані та анулює гарантію.

Samsung — коротка інструкція

  1. Параметри розробника → увімкнути OEM Unlock.
  2. Вимкнути телефон → Volume Down + Bixby/Side + Power → Download Mode.
  3. Використовувати Odin3 або офіційні Samsung інструменти.

Деякі моделі операторів не дозволяють розблокування.

💾 Як встановити TWRP

  1. Завантажити підходящий twrp.img для вашої моделі.
  2. Fastboot → 
    fastboot flash recovery twrp.img
fastboot reboot recovery
  3. Wipe → Advanced Wipe (system/data/dalvik) перед встановленням ROM / GApps.

A/B пристрої — використовуй slot інструкцію.

🔌 Драйвери ADB / Fastboot

  • Android Platform Tools (adb, fastboot)
  • USB-драйвери: Xiaomi, MediaTek, Qualcomm, Samsung
  • Перевірка: 
    adb devices
fastboot devices

🧩 microG / GApps

Для Android 15 офіційних GApps немає — microG дає альтернативу. Використовуй Magisk-модуль microG Installer Revived або APK вручну.
