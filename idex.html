Поширені помилки під час прошивки

1. Помилка «The system has been destroyed» (Xiaomi)

Виникає, якщо встановлена неправильна прошивка або пошкоджений boot/recovery.

Рішення:

  • Прошити повну Fastboot-прошивку через MiFlash
  • Перевірити, що прошивка під ваш регіон і модель

2. Помилка «Пінгвін» (Linux recovery)

Телефон входить у режим Qualcomm/MTK або не може знайти систему.

Рішення:

  • Перепрошити через Fastboot або SP Flash Tool
  • Відновити розділ boot або system

Як встановити драйвери ADB та Fastboot

  1. Завантажити Minimal ADB & Fastboot або офіційні драйвери Google.
  2. Увімкнути «Налагодження по USB» у параметрах розробника.
  3. Підключити телефон і дозволити USB Debugging.
  4. Перевірити командою: 
    adb devices
  5. Щоб перевірити Fastboot: 
    fastboot devices

Розблокування завантажувача (Xiaomi)

  1. Увімкнути «Статус розблокування Mi» у параметрах розробника.
  2. Завантажити офіційний Mi Unlock Tool.
  3. Увійти в Fastboot режим (гучність - + живлення).
  4. Підключити до ПК та натиснути Unlock.
  5. Дочекатися таймера (72 години або 168 годин).

Як встановити TWRP (Xiaomi)

  1. Завантажити правильний twrp.img для вашої моделі.
  2. Покласти файл у папку ADB.
  3. Перезавантажити телефон у Fastboot режим.
  4. Ввести команду: 
    fastboot flash recovery twrp.img
  5. Одразу перезавантажити у рекавері: 
    fastboot reboot recovery

Прошивка Samsung та встановлення TWRP

Як увійти в режим завантаження (Download Mode)

  1. Вимкніть телефон.
  2. Затисніть одночасно кнопки Гучність вниз + Домашня (Home) + Живлення для старих моделей, або Гучність вниз + Бік + Живлення для нових моделей без Home.
  3. На екрані з'явиться режим Download Mode, підтвердіть кнопкою Гучність вгору.

Як прошити Samsung через Odin3

  1. Запустіть Odin3 на ПК.
  2. Підключіть телефон у режимі Download Mode.
  3. У Odin3 виберіть відповідний файл прошивки (AP, BL, CP, CSC).
  4. Натисніть Start і дочекайтеся завершення процесу.

Як встановити TWRP на Samsung

  1. Завантажте файл TWRP для вашої моделі Samsung (.tar).
  2. Запустіть Odin3, підключіть телефон у Download Mode.
  3. У Odin3 виберіть TWRP файл у полі AP.
  4. Натисніть Start і дочекайтеся завершення прошивки.
  5. Після завершення вимкніть телефон і увійдіть у TWRP рекавері (Volume Up + Бік + Home).
