🔧 Розблокування завантажувача

Xiaomi — швидкі кроки

  1. Налаштування → Про пристрій → натиснути 7 разів на "Номер збірки", потім в "Параметрах розробника" увімкнути OEM Unlock і USB Debugging.
  2. Використати офіційний Mi Unlock Tool на ПК (логін у Mi Account потрібен).
  3. Phone → Fastboot (Vol− + Power) → підключити до ПК → Run Mi Unlock → Unlock. Час очікування може бути 72–168 годин.

Увага: розблокування стирає всі дані та анулює гарантію.

Samsung — коротка інструкція

  1. Параметри розробника → увімкнути OEM Unlock (якщо доступно).
  2. Вимкнути телефон → натиснути Volume Down + Bixby/Side + Power (або для старих: Volume Down + Home + Power) → підтвердити перехід в Download Mode.
  3. Використовувати Odin3 для прошивки або офіційні інструменти Samsung.

Деякі моделі операторів не дозволяють розблокування.

💾 Як встановити TWRP

  1. Завантажити підходящий twrp.img для вашої моделі.
  2. Підключити телефон в Fastboot → виконати: 
    fastboot flash recovery twrp.img
fastboot reboot recovery
  3. У TWRP зробити Wipe → Advanced Wipe (system/data/dalvik) перед встановленням ROM або GApps.

Якщо пристрій A/B — використовуй інструкцію для A/B (slot). Не дозволяй системі перезаписувати recovery після інсталяції.

🔌 Драйвери ADB / Fastboot

  • Встанови Android Platform Tools (adb, fastboot).
  • Постав USB-драйвери: Xiaomi, MediaTek, Qualcomm, Samsung USB Driver.
  • Перевір: 
    adb devices
fastboot devices

🧩 microG / GApps (коротко)

На Android 15 офіційних GApps часто немає — microG дає альтернативу. Для встановлення рекомендується використовувати Magisk-модуль microG Installer Revived або встановлювати APK-кроки вручну.
