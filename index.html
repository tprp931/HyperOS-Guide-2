Цей гайд допоможе розблокувати завантажувач, встановити драйвери, зрозуміти системні режими та безпечно прошивати смартфон.

📱 Що таке Recovery?

Recovery — це спеціальний режим смартфона, де можна скидати налаштування, встановлювати прошивки, робити wipe або резервні копії.

  • Stock Recovery — стандартне, обмежене.
  • Custom Recovery — наприклад TWRP, має набагато більше функцій.

⚠ Поширені помилки

  • The system has been destroyed — неправильна прошивка.
  • Bootloop — зависання на логотипі.
  • No command — стандартне recovery.

🔓 Розблокування Bootloader

📌 Xiaomi (повна інструкція)

  1. Налаштування → Про телефон → 7 разів на «Версія MIUI».
  2. Увімкнути: USB Debugging OEM Unlock
  3. Прив'язати Mi Account:
  4. Fastboot → ПК → Mi Unlock Tool.
  5. Очікування від 72 до 168 годин.
  6. Натиснути Unlock.

📌 Samsung (повна інструкція)

  1. Увімкнути OEM Unlock.
  2. Зайти в Download Mode:
    Vol- + Bixby + Power або Vol- + Home + Power
  3. Утримувати Vol+ щоб підтвердити unlock.
  4. Прошити через Odin (BL, AP, CSC).
