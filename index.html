Розблокування TWRP Драйвери microG / GApps Відео

🔧 Розблокування завантажувача

Xiaomi — швидкі кроки

  1. Увімкнути OEM Unlock та USB Debugging.
  2. Використати Mi Unlock Tool на ПК.
  3. Fastboot → Mi Unlock → очікування 72–168 годин.

⚠ Стирає всі дані, анулює гарантію.

Samsung — коротка інструкція

  1. Увімкнути OEM Unlock у налаштуваннях.
  2. Перезавантажити в Download Mode.
  3. Прошити через Odin3.

⚠ Деякі моделі операторів заблоковані.

💾 Як встановити TWRP

  1. Завантажити правильний twrp.img.
  2. Fastboot: 
    fastboot flash recovery twrp.img
fastboot reboot recovery
  3. Wipe → system/data/dalvik перед прошивкою ROM/GApps.

A/B пристрої потребують інструкції по слотам.

🔌 Драйвери ADB / Fastboot

  • Android Platform Tools
  • USB-драйвери: Xiaomi, MediaTek, Qualcomm, Samsung
  • Перевірка: 
    adb devices
fastboot devices

🧩 microG / GApps

Для Android 15 поки немає офіційних GApps — використовуй microG або Magisk модуль microG Installer Revived.
